# Проект Space
Учебный репозиторий для практической работы №5 по Git.
## Цель
Освоить работу с GitHub: создание репозитория, загрузка файлов, ветвление, Pull Request и форк.
## Структура
- **main** — базовые материалы (скрипт, схема, отчёт, README).  
- **ВтораяВетвь** — дополнительные файлы (данные, таблица).  
- **ТретьяВетвь** — текстовый файл «Назначение СКВ.txt».  
## Примечание
Форк внешнего проекта [pytorch/pytorch](https://github.com/pytorch/pytorch) выполнен для демонстрации работы с сообществом.
